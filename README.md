# Car Sales Dashboard 

This project analyzes car sales data using Microsoft Excel.

## Key Steps:
- Cleaned raw data (removed duplicates, handled missing values, formatted columns)
- Created Pivot Tables for summarizing sales data
- Built charts to visualize trends and performance
- Designed a dashboard for easy understanding

## Key Insights:
- Monthly sales trends
- Top-selling car models
- Region-wise sales performance

## Tools Used:
- Microsoft Excel
- Pivot Tables
- Charts
